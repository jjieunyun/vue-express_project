# frontend

## Project setup
```
cd frontend
npm install

cd backend
npm istall
```

### 반드시 한번은 SET DEBUG=backend:* npm start 실행
```
SET DEBUG=backend:* npm start
or
node ./bin/www
or
nodemon ./bin/www
```



### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
